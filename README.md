# Linux-practice-v1

## Practice Linux command 
---
| Letter | Command   | Description                                      |
|--------|-----------|--------------------------------------------------|
| A      | `alias`   | Create shortcuts for commands                    |
| B      | `bg`      | Resume a suspended job in the background         |
| C      | `cat`     | Display file contents                            |
| D      | `df`      | Show disk space usage                            |
| E      | `echo`    | Print messages to the terminal                   |
| F      | `find`    | Search for files and directories                 |
| G      | `grep`    | Search text using patterns                       |
| H      | `history` | Show previously executed commands                |
| I      | `ifconfig`| Configure or display network interfaces          |
| J      | `jobs`    | List active jobs in the current shell session    |
| K      | `kill`    | Terminate processes by PID                       |
| L      | `ls`      | List directory contents                          |
| M      | `man`     | Display manual pages for commands                |
| N      | `netstat` | Show network connections and statistics          |
| O      | `open`    | Open files with default applications (GUI-based) |
| P      | `ps`      | Display currently running processes              |
| Q      | `quota`   | Show user disk usage and limits                  |
| R      | `rm`      | Remove files or directories                      |
| S      | `ssh`     | Secure shell for remote login                    |
| T      | `top`     | Display real-time system processes               |
| U      | `uname`   | Show system information                          |
| V      | `vi`      | Launch the `vi` text editor                      |
| W      | `whoami`  | Show current logged-in user                      |
| X      | `xargs`   | Build and execute command lines from input       |
| Y      | `yum`     | Package manager for RPM-based systems            |
| Z      | `zip`     | Compress files into a zip archive                |
