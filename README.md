# Linux-practice-v1

## üêß A to Z Linux Commands Cheat Sheet
---

| Letter | Command     | Description                                      | Example Usage                                      |
|--------|-------------|--------------------------------------------------|----------------------------------------------------|
| A      | `alias`     | Create shortcuts for commands                    | `alias ll='ls -la'`                                |
| B      | `bg`        | Resume a suspended job in the background         | `bg %1`                                            |
| C      | `cat`       | Display file contents                            | `cat file.txt`                                     |
| D      | `df`        | Show disk space usage                            | `df -h`                                            |
| E      | `echo`      | Print messages to the terminal                   | `echo "Hello, Linux!"`                             |
| F      | `find`      | Search for files and directories                 | `find /home -name "*.txt"`                         |
| G      | `grep`      | Search text using patterns                       | `grep "error" logfile.txt`                         |
| H      | `history`   | Show previously executed commands                | `history | grep ssh`                               |
| I      | `ip`        | Show/manipulate routing, devices, policy         | `ip addr show`                                     |
| J      | `jobs`      | List active jobs in the current shell session    | `jobs`                                             |
| K      | `kill`      | Terminate processes by PID                       | `kill -9 1234`                                     |
| L      | `ls`        | List directory contents                          | `ls -l /var/log`                                   |
| M      | `man`       | Display manual pages for commands                | `man grep`                                         |
| N      | `nano`      | Simple terminal text editor                      | `nano config.txt`                                  |
| O      | `openssl`   | Toolkit for SSL/TLS and cryptography             | `openssl genrsa -out key.pem 2048`                 |
| P      | `ps`        | Display currently running processes              | `ps aux | grep nginx`                              |
| Q      | `quota`     | Show user disk usage and limits                  | `quota -v`                                         |
| R      | `rm`        | Remove files or directories                      | `rm -rf /tmp/testdir`                              |
| S      | `ssh`       | Secure shell for remote login                    | `ssh user@192.168.1.10`                            |
| T      | `top`       | Display real-time system processes               | `top`                                              |
| U      | `uname`     | Show system information                          | `uname -a`                                         |
| V      | `vim`       | Advanced terminal text editor                    | `vim ~/.bashrc`                                    |
| W      | `whoami`    | Show current logged-in user                      | `whoami`                                           |
| X      | `xargs`     | Build and execute command lines from input       | `cat files.txt | xargs rm`                         |
| Y      | `yum`       | Package manager for RPM-based systems            | `yum install httpd`                                |
| Z      | `zip`       | Compress files into a zip archive                | `zip archive.zip file1 file2`                      |
